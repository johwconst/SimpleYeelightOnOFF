## About

This is a very simple CRUD to turn on / off the Xiaomi Yeelight lamp ðŸ’¡ in Windows, implementation of the [Yeelight Python](https://gitlab.com/stavros/python-yeelight) with an interface using [Tkinter](https://docs.python.org/3/library/tk.html)


## Setup

It is necessary that your lamp is configured for management via LAN, After clone install the necessary applications using the command:

```
pip install -r requirements.txt
```


## Run

To run use:

```
python main.py
```
